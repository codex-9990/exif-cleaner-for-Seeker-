# Exif Cleaner for Seeker

أداة محمولة تركز على الخصوصية مصممة لإزالة البيانات الوصفية (بيانات Exif) من صورك قبل مشاركتها عبر الإنترنت.

## لماذا المصدر المفتوح؟

**خصوصيتك هي أولويتنا.**
نؤمن بأن الأدوات التي تتعامل مع البيانات الحساسة يجب أن تكون شفافة. بجعل هذا المشروع مفتوح المصدر، نسمح لأي شخص بفحص الكود والتحقق من أن:
- **لا يتم تحميل أي بيانات:** تتم جميع المعالجات محليًا بنسبة 100٪ على جهازك.
- **لا يوجد تتبع مخفي:** نقوم ببساطة بإزالة البيانات الوصفية وحفظ الصورة النظيفة مرة أخرى في مكتبتك.

## الميزات

- **إزالة بيانات Exif:** يزيل موقع GPS وإعدادات الكاميرا والبيانات الوصفية الأخرى.
- **المعالجة دون اتصال:** لا يلزم الاتصال بالإنترنت. صورك لا تغادر هاتفك أبدًا.
- **المعالجة المجمعة:** حدد ونظف صورًا متعددة في وقت واحد.
- **دعم متعدد اللغات:** الإنجليزية، اليابانية، الفرنسية، الألمانية، الإيطالية، الإسبانية، الصينية (التقليدية / المبسطة)، الكورية، العربية.
- **الوضع الداكن:** مدعوم بالكامل.
- **دعم المطور:** خيار التبرع بـ SOL لدعم المشروع.

## كيف يعمل

1. اختر الصور من معرض الصور الخاص بك.
2. ينشئ التطبيق نسخة نظيفة بدون بيانات وصفية.
3. احفظ النسخة النظيفة في معرض الصور الخاص بك أو شاركها مباشرة.

## التطوير

تم بناء هذا المشروع باستخدام [Expo](https://expo.dev) و React Native.

```bash
# تثبيت التبعيات
npm install

# بدء التطبيق
npx expo start
```

## البناء (Build)

### Android

```bash
# Preview build (APK للاختبار)
npx eas-cli build -p android --profile preview

# Production build (AAB لمتجر Google Play)
npx eas-cli build -p android --profile production
```

> **ملاحظة لمستخدمي Windows:**
> إذا واجهت خطأ أمان PowerShell، أضف `cmd /c` قبل الأمر:
> ```powershell
> cmd /c npx eas-cli build -p android --profile preview
> ```
